WorkshopOS

AI 多专家多轮深度研讨会工作台。

付费 · 企业部署 多模型混搭 本地运行

技术栈

后端：Python + FastAPI + SSE 实时流
引擎：队列驱动状态机 + JSON 三层兜底解析
模型：OpenAI 兼容，支持 Qwen / DeepSeek / GLM / GPT 任意混搭

核心能力

🎤

主持人开场

AI 主持人介绍议题、规则、专家背景。

💡

多轮深度研讨

逐轮深入，每轮自动提炼焦点问题。

🤝

共识 / 分歧提炼

每轮结束自动抽取各方共识与待议事项。

🗳️

投票表决

对关键争议点进行 AI 专家投票。

📝

结构化纪要

自动生成含 N 轮脉络的会议纪要 + Markdown 导出。

🔀

多模型混搭

每个专家可以用不同的模型 / API / 人设。

适用场景

  1. 咨询公司 / 战略部门重大决策前的多视角预研讨，降低决策盲区。
  2. 投资委员会投资标的的多维度评估，模拟不同专家立场。
  3. 学术研讨 / 课题研究多学派观点碰撞，快速生成研讨框架。
  4. 企业内部头脑风暴跨部门议题的模拟圆桌讨论。

特点：纯本地运行，78 项测试全过，数据不上云。可按需定制专家人设、研讨轮次、输出格式。
