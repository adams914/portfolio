CMS · 合同管理

扫描件丢进去，AI 帮你管合同。

付费 · 企业部署 桌面应用

技术栈

框架：Tauri2 + Vue3 + Element Plus
后端：Rust，跨平台桌面运行时
AI：PaddleOCR + LLM 自动提取
数据库：SQLite，本地优先存储

核心功能

📥

热文件夹自动入库

扫描件丢进去就自动处理，无需手动操作。

🔍

OCR 识别

基于 PaddleOCR，扫描件变成可搜索文本。

🤖

LLM 自动提取

合同金额、签订方、到期日、关键条款自动结构化。

🔎

自然语言搜索

“去年和甲方签的合同”、“快到期的”都能搜。

⏰

到期提醒

合同临期自动预警，不再错过续签。

📄

PDF 预览

不打开原件也能翻看合同内容。

💾

本地优先

数据不上云，合同信息安全可控。

使用方法

  1. 安装桌面应用下载安装包，双击即可运行。
  2. 丢入合同扫描件把 PDF / 图片放入指定热文件夹。
  3. 自动 OCR + AI 提取系统自动识别文本并提取关键信息入库。
  4. 自然语言搜索用一句话搜到任意合同，按条件过滤。

特点：扫描件 → 可搜索的智能合同库。
