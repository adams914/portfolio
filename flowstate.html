使用方法

  1. 下载安装包从 GitHub 下载 flowstate_1.0.0_x64-setup.exe（约 2.5MB）。
  2. 安装并启动双击安装，完成后启动 Flowstate。
  3. 直接开始使用无需注册账号，打开即用。
  4. 清空大脑把所有待办丢进收件箱，再逐步分类处理到对应清单。
