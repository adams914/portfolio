好记 HaoMark

轻量、快速、实时渲染的 Markdown 编辑器。v2.1 Notion 柔和风。

桌面应用 · Windows v2.1.1 仅 4.9MB 免费

核心功能

⚡

实时渲染

Obsidian 风格，光标行源码、其余行自动渲染。点击代码块/表格/公式即时编辑。

📑

多标签页

Chrome 风，同时打开多个文档，每 tab 独立光标/撤销/滚动。关闭重开自动恢复。

🎨

Notion 柔和风 UI

v2.1 重设计，暖灰背景 + SVG 图标 + 凹槽式视图切换，写作感优先。

🌲

CodeMirror 语法树

v2.0 渲染核心重写，根治行内代码/标题强调/代码围栏三类解析问题。

📐

完整渲染能力

GFM 全语法 + 代码高亮 + KaTeX 数学公式 + Mermaid 流程图。

🔒

数据安全

保存原子写入（断电不损坏）、GBK/UTF-16 自动识别、单实例锁。

🎛️

设置面板

基础/编辑/渲染/应用四类偏好，即时生效，自动持久化。

🪶

极轻量

安装包仅 4.9MB，.md 文件双击秒开。

使用方法

  1. 下载安装包从 GitHub 下载好记安装包（约 4.9MB）。
  2. 双击打开安装后双击任意 .md 文件即可用编辑器打开。
  3. 或新建文档也可以在应用内直接创建新文档开始写作。
  4. 高效编辑用快捷键高效编辑，实时预览排版效果。

对标 Obsidian 的实时渲染体验，但安装包只有 4.9MB。v2.1 新增 Notion 柔和风 UI 重设计。
