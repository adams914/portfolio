好找 v2.0

本地优先的智能文档管理工具。

桌面应用 · Windows 10/11 免费

核心功能

📥

一键导入文件夹

整个目录拖进来，自动识别 PDF / Word / PPT / Excel。

🔍

全文搜索

支持 AND / OR 语法，跨文档秒级检索。

📷

OCR 扫描件识别

基于 PaddleOCR-VL，扫描版 PDF 也能搜。

🔁

重复检测

按文件哈希秒级找出重复文档。

🏷️

规则自动分类

规则优先，省下 90% 的 AI 成本。

⚡

批量 AI 队列

内置限速保护，专为免费 API 设计。

🔀

AI 标签 / 分类合并

把 AI 结果一键合并进文档库。

📄

不打开就能看

PDF 可翻页、Word 可预览，免去来回切换。

使用方法

  1. 下载从 GitHub Releases 下载 好找_v2.0.0_windows_x64.zip。
  2. 解压并启动解压到任意位置，双击 好找.exe 即可运行。
  3. 导入文档点「📁 目录」选择文件夹，把文档一次性导入。
  4. 配置 AI 并整理设置 → 配置 AI 服务（推荐硅基流动）→ 开始批量整理。

系统要求：Windows 10 / 11（64 位）。
